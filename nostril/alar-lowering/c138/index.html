1. 비공내리기

김상호 원장 · 비공내리기 · 로코코성형외과

1. 비공내리기

선천적으로 콧구멍을 덮고 있는 연부조직이 부족하여 정면에서 콧구멍이 보이는 분들이 있습니다.

이렇게 콧구멍이 노출되면 미관상으로도 보기가 좋지 않고

관상학적으로도 재물이 모이지 않는 상 이라 하여 교정을 원하시는 고객분들이 많답니다.

이를 교정하는 수술법은 그동안 학회에서 세계적으로 꾸준히 발표되었고,

개인적으로도 다양한 수술법을 적용해 보았습니다.

그 결과 현재 사용하는 수술법이 가장 결과가 좋다는 것을 알게 되었고,

이 수술로 생길 수 있는 다양한 문제점들을 극복하는 방법도 오랜 기간에 걸쳐 터득하게 되었습니다.

이러한 수술법은 코성형을 전문으로 하는 병원 중에서도 일부 병원에서만 가능한 수술법이랍니다.

2013.8.11

안녕하세요.

김상호원장입니다.

오늘은 비공내리기 (콧날개내리기, M자 콧구멍 교정술) 수술에 대해 좀 더 알아보겠습니다.

정상적인 콧구멍은 정면에서 대략 이정도 모습입니다.

선천적으로 코끝의 날개연골 일부의 발달이 저하되어 있고, 콧날개쪽 연부조직이 부족한 경우에

콧구멍이 삼각형모양으로 보이는 경우가 있습니다.

양측을 같이 보면 알파벳 M 자 모양으로 보이기 때문에 저는 개인적으로 M자 콧구멍이라고 부릅니다.

대략 다음과 같은 모습의 콧구멍을 말합니다.

콧구멍만 올라간 경우도 있고, 심한 경우는 콧날개와 코끝의 연결부위가 움푹 꺼져서 집게로 집어놓은 것 처럼 보이기도 합니다.

이런 M자 콧구멍은 비중격 혹은 늑연골을 이용해서 코끝을 높이는 경우(고정형 지지대 사용시), 혹은 어떤 방식의 코성형이든 날개연골을 묶어주는 경우에 코끝이 높아질 수록 비공(콧구멍)이 노출되는 정도가 심해지고, 코끝이 찝혀보이게 된답니다.

아래쪽에서 보는 경우 동그란 콧구멍에서 코끝을 높인 후 강낭콩모양의 서양형 콧구멍으로 변할수록 정면에서는 콧구멍이 더 많이 보이게 되는 것이지요.

수술 후 비공이 노출되는 정도는 날개연골의 특성, 연부조직의 특성, 수술법에 따라 그 정도에 차이가 있을 수 있지만, 거의 대부분의 경우 고정형 코끝높임술을 사용하는 경우 약간이라도 비공이 더 노출된답니다.

수술법 중 코끝까지 실리콘을 넣고 연골등으로 끝부분을 감싸주는 방식의 수술에서는 이렇게 수술 후 비공이 노출되는 정도가 매우 적습니다.

M자 콧구멍이 되었다고 해서 모든 경우에 미워보이는 것은 아닙니다.

콧구멍의 바닥쪽 형태에 따라 콧구멍이 더 보일수도 있고,

적당히 콧구멍이 노출되면서 코끝이 작아보이는 장점을 얻는 경우도 있답니다.

즉, 모든 M자 콧구멍이 교정을 필요로 하는 것은 아닙니다.

콧구멍이 많이 보여도 전혀 신경쓰지 않고 잘 지내시는 분들도 계시고, (a급 연예인들 중에도 이런 경우가 있습니다)

콧구멍이 살짝만 보이는데도 매우 싫어하시는 분들도 계십니다.

가장 중요한 것은 본인의 주관적인 판단입니다.

M 자 콧구멍은 크게 세가지 type 으로 나눌 수 있습니다.

콧구멍이 올라간 끝부분의 위치가 안쪽이냐, 중간이냐, 바깥쪽이냐에 따라 type 1,2,3 으로 나누고 있습니다.

사진으로 보시면

(1) Type 1

M자 콧구멍 Type 1 은 콧구멍이 위쪽으로 뾰족한 부분이 안쪽에 몰려있는 경우입니다.

세가지 type중에서 비공내리기수술의 효과가 가장 좋은 Type 입니다.

가장 일반적인 type 1 M자 콧구멍입니다. 이전 타병원에서 귀연골과 실리콘으로 두차례 코성형을 하신 상태입니다.

같은 type1 비공이지만 위쪽 사진과의 차이를 보시면 비공의 바닥쪽이 아래쪽으로 뾰족한 w 자 모양을 만들어 훨씬 더 콧구멍이 많이 보이게 됩니다.

그리고 콧구멍이 위쪽으로 노출된 정도도 위의 사진에 비해 심한 편이며, 콧날개가 코끝과 연결되는 부위가 꺼져서 찝혀보이는 모습을 확인할 수 있습니다.

좌측이 더 심한 상태입니다. (많은 경우에 우측보다 좌측이 더 꺼져 보이는 것을 관찰할 수 있습니다)

타병원에서 수차례 수술을 받으시고 들린코와 비공이 보이는 것이 교정되지 않은 상태에서 염증으로 구축까지 동반된 경우입니다.

Type 1 비공인데, 얼핏보면 모르지만 자세히 보면 콧날개와 비공의 크기 모두 상당한 비대칭이 보입니다.

이러한 비대칭은 교정에 한계가 있답니다.

역시 타병원에서 수차례 늑연골 등으로 재수술을 받으신 상태입니다.

더 심한 비대칭을 보이는 type 1 비공입니다.

두번째 사진처럼 비공의 바닥이 w모양을 보이고 있습니다.

타병원에서 비중격, 귀연골, 기증늑연골 등으로 수차례 수술을 받으신 상태입니다.

역시 심한 비대칭을 가진 type1 비공입니다.

콧날개의 높이 자체가 심한 차이를 보입니다.

단순한 콧날개의 비대칭이 아닌 안면비대칭을 가진 경우로 좌측얼굴이 전체적으로 우측보다 높은 경우입니다.

이런경우는 비공내리기만으로는 교정에 한계가 있고, 콧날개 올리기 수술을 같이 해서 어느정도

비대칭을 보정하게 됩니다.

타병원 코성형 후 염증성 구축으로 들린코가 진행된 상태입니다.

(2) Type2

M자 콧구멍 Type 2는 콧구멍이 위쪽으로 뾰족한 부분이 중앙부에 위치한 경우입니다.

세가지 type중에서 비공내리기수술의 효과가 중간인 Type 입니다.

심한 들린코에 동반된 type 2 비공입니다.

수술전 상태이며, 사진과 같은 선천적인 들린코의 경우 피부조직이 모자라서 들린코 교정이 더욱 어렵습니다.

전형적인 type 2 비공입니다.

타병원에서 수술받으신 분으로 비공내리기 수술을 받으셨지만 효과를 보지 못하셔서 비공내리기 수술만을 위해 내원하신 경우입니다.

비공내리기(콧날개 내리기, 콧구멍 가리기) 수술방법은 매우 많은 수술법들이 교과서와 논문으로 소개되어 있습니다.

저도 여러가지 수술법을 사용해 보았지만 그 중에서 현재 제가 사용하는 방법이 가장 확실한 효과를 볼 수 있다고 생각합니다.

Type2 비공에서 비대칭이 동반된 경우입니다.

사진에서는 잘 보이지 않지만 비주비대칭(비중격만곡증)이 동반된 상태입니다.

타병원에서 수차례 수술받으신 상태로 좌측 콧날개가 내려앉으면서 찝혀보이는 상태입니다.

(3) Type 3

M자 콧구멍 Type 3 은 콧구멍이 위쪽으로 뾰족한 부분이 바깥쪽으로 벌어져 있는 경우입니다.

세가지 type중에서 비공내리기수술의 효과가 가장 떨어지는 Type 입니다.

전형적인 type 3 비공입니다.

M자의 첨단부가 너무 바깥쪽에 위치하여 비공을 가리는 경우에 안쪽 살들이 같이 내려오면서 그리 예쁜 느낌이 아닌 결과가 나오게 됩니다.

역시 type 3 비공입니다.

type 3 비공에 비공, 콧날개 비대칭과 동시에 비중격, 안면비대칭이 동반된 복합비대칭 상태입니다.

콧구멍을 어느정도 가리는 것을 목표로 해서 수술하며, 비대칭의 교정은 한계가 있습니다.

콧구멍을 맞추면 정면에서 콧대가 비뚤어지는 안면비대칭 상태이므로 이런경우는 콧대가 일자로 보이는 것을 우선으로 합니다.

더 큰 것(일자 콧대)를 얻기 위해 작은 것(비공 콧날개 비대칭)을 버리는 것이지요.

타병원에서 실리콘 귀연골, 비중격 연골 등으로 수차례 수술받으신 상태입니다.

심한 Type 3 비공에 찝힌느낌도 심한상태입니다.

코끝의 날개연골발육이 매우 부족한 상태로 연골과 연부조직의 보충이 동시에 충분히 이루어져야 합니다.

타병원에서 비중격연골 이용해서 코성형받으신 상태입니다.

그럼 각각의 타입에서 비공내리기(콧날개 내리기) 수술로 교정한 모습을 같이 보시겠습니다.

(1) Type 1

이전 수술 후 들린코에 콧구멍이 보이는 상태입니다.

자가 늑연골 이용해서 들린코를 교정하면서 동시에 비공내리기 시행한 상태입니다.

들린코만 교정되면 콧구멍도 가려질 것이라고 생각하시는 분들이 많으신데요,

들린코가 교정되어도 콧구멍은 여전히 보이게 된답니다.

이전 수술했던 분 중에 콧구멍이 보이는 것은 신경쓰지 않으니 들린코와 구축코끝 교정만 해달라고 하셨던 분이 계셨습니다.

약간의 들린코를 자가늑연골 이용해서 교정한 상태입니다.

이전 염증으로 코끝으로 보형물이 뚫고나와 생긴 구축부위가 잘 교정된 것을 볼 수 있습니다.

사진에서 보는 정도라면 비공내리기 수술을 같이 받으셨다면 더 좋은 결과를 만들 수 있었을 거란 생각이 듭니다.

하지만 수술받으신 분은 저 상태로 충분히 만족하셨답니다.

미용성형은 자기 만족이 가장 중요합니다.

위에 사진과 비교해보시면 비공내리기를 한 것과 하지 않은 결과의 차이를 아실 수 있겠지요?

위에 수술받으신 분의 코를 아래쪽에서 찍은 사진입니다.

생각보다 많은 분들이 수술전 사진에서처럼 코끝에 구멍이 나서 구축이 오면서 쏙 꺼진 상태로 상담을 오시는데요,(대부분 염증에 의해서 생김)

정성껏 수술하면 사진처럼 좋은 결과를 얻을 수 있답니다.

역시 염증에 의한 구축성 들린코와 비공노출이 동반된 경우입니다.

tyoe 1 비공상태인데요,

자가늑연골이용해서 구축을 풀어주면서 들린코를 교정하고

비공내리기수술을 같이 시행해서 좋은 결과를 얻을 수 있었습니다.

비공내리기 수술의 장점으로 콧구멍을 가리면서 동시에 찝힌코끝을 교정할 수 있습니다.

수술전 좌측의 찝힌 느낌이 자연스럽게 교정된 것을 보실 수 있습니다.

이전 많은 코성형을 받으신 상태로 늑연골도 몇번 사용하셨던 분입니다.

다시 자가늑연골 이용해서 코끝을 만들어 주고, 동시에 비공내리기 시행하여 콧구멍을 가렸습니다.

수술 후 편안한 느낌의 코끝이 되었습니다.

이전 비중격, 귀연골, 늑연골로 수술받으셨지만 들린코와 심한 비공비대칭이 남은 분입니다.

자가늑연골로 코끝휜것 교정하면서 들린코 교정하고, 비공내리기 시행하였습니다.

전체적으로 뒤틀어진 양상의 코끝이 단정한 모양으로 바뀌었습니다.

비공비대칭이 심한경우에 완전 교정은 힘들답니다. 다만 어느정도 비슷한 모양으로 만들 수는 있습니다.

이전 수차례 코성형 후 만성염증으로 인한 구축성 들린코상태입니다.

안면비대칭이 심하신 편이라 휜느낌은 교정해도 어쩔 수 없이 어느정도 남게 됩니다.

오래전 수술한 case라 사진상태가 좋지 않습니다.

자가늑연골로 구축풀어주면서 들린코를 교정하고 비공내리기 하였습니다.

Type1 비공으로 비공내리기의 결과가 좋은편입니다.

우측 콧날개가 많이 쳐져있어 콧날개 올리기 같이 시행하여 비대칭을 최대한 교정하였습니다.

(2) Type 2

type 2 비공입니다.

선천성 들린코로 첫수술에는 늑연골을 잘 사용하지 않지만 코가 짧고 들려있는 경우는 비중격도 그 크기가 작아서 충분히 들린코를 교정하기에 모자랍니다.

자가늑연골 이용해서 최대한 코끝을 높이면서 들린코를 교정하였고, 동시에 비공내리기로 콧구멍을 가렸습니다.

타병원에서 코재수술과 함께 비공내리기 수술을 받으셨지만 비공내리기 수술결과에 만족하지 못하셔서 내원하셨습니다.

추가적인 비공내리기 수술만 단독으로 시행한 경우입니다.

type 2 비공으로 type 1 에 비해서는 약간 효과가 떨어집니다.

코끝 모양이 마음에 들지 않아 타병원에서 수차례 재수술을 받으셨지만 M자 콧구멍과 집힌 느낌 때문에 로코코성형외과를 찾으신 분입니다.

전형적인 M자 콧구멍에 찝힌느낌이 동반된 경우입니다.

비공내리기 수술로 뾰족한 콧구멍과 찝힌느낌을 동시에 교정할 수 있습니다.

최대한 비대칭 교정하였지만 콧날개 자체의 볼룸차이나 높이차이를 극복하는 데에는 어느정도 한계가 있습니다.

(3) Type 3

M자 콧구멍이 바깥쪽으로 벌어진 형태라 비공내리기를 했을 때 모양이 썩 예쁘게 나오지 않습니다.

비공이 가려지는 정도도 다른 type 에 비해 덜 가려지는 편이랍니다.

콧구멍이 가려지긴 했지만 뭔가 약간의 아쉬움이 있는데요,

구조적으로 저런정도의 결과가 나올 수 밖에 없답니다.

앞으로 더 연구해서 보다 나은 결과를 만들 수 있도록 노력하겠습니다.

심한 비대칭과 화난 눈썹처럼 바깥쪽이 치켜올라간 type 3 비공입니다.

자가늑연골로 코끝 교정하고 비공내리기수술로 편안한 느낌의 코끝으로 바뀌었습니다.

비중격이나 늑연골 이용해서 코끝을 높였을 때 생길 수 있는 문제점입니다.

애초에 날개연골의 볼룸이 약하고, M자 콕구멍을 가진 코에서 특별한 계획없이 코끝을 높이게 되면 사진의 수술전 코처럼 심한 M자 콧구멍과 찝힌 느낌의 코끝이 될 수 있습니다.

코끝이 너무 좁고 수술한 티가 나는 느낌을 없애기를 원하셨습니다.

비공내리기 수술로 콧구멍을 어느정도 가리면서 코끝에 볼룸을 줘서 부드러운 느낌으로 바뀌었습니다.

아직 붓기가 남은 상태의 코끝이라 시간이 지나면서 붓기가 조금 더 빠지게 됩니다.
