Rear Cheekbone Reduction

변형 15분 광대축소 ( feat. 퀵광대 ) + 뒷광대축소

지금 생각에는 퀵광대 방식의 수술중에서는 현재 집도중인 수술에서 더 이상은 큰 변화가 없을 것 같다는 생각이 들만큼 어느 정도 완성된 수술이 되었다고 생각하고 있습니다.

김상호 원장 · 뒷광대축소술 · 로코코성형외과

변형15분광대축소 ( feat. 퀵광대 ) + 뒷광대축소

안녕하세요.

도와줘요 로코코~ 김상호원장입니다.

13년전쯤?

퀵광대를 처음 집도할 때만 해도

전국에 퀵광대축소술 방식으로 수술하는 병원이

3군데? 정도 있었습니다.

다들 조금 다른 방식으로 수술을 하셨었는데요,

제 나름대로 연구해서

"15분 광대축소술"

이란 이름을 걸고 수 많은 분들께

집도해드렸네요.

이후에도 단순 퀵광대 방식의 옆광대축소에서

45도 광대뼈까지 같이 볼룸을 줄이는

변형15분 광대축소술 로 발전시켰고,

아쉬운 부분을 계속해서

개선시키다보니

광대라인 지방흡입술,

미니리프팅,

심부볼 지방이식,

관자지방이식,

뒷광대축소

등의 수술들을

연구개발하고 집도하게 되었습니다.

지금 생각에는

퀵광대 방식의 수술 중에서는

현재 집도중인 수술에서

더 이상은 큰 변화가 없을 것 같다는 생각이 들만큼

어느 정도 완성된 수술이 되었다고

생각하고 있습니다.

아무리 생각해도..

정말 좋은 수술법입니다.

요즘 많이 집도하는 수술 중에서

변형15분광대축소술 + 뒷광대축소술

case 보여드리겠습니다.

수술 전 사진입니다.

이전 뒷광대 포스팅에서 여러번 언급했듯이

사진에서처럼 뒷광대 부위가

혹처럼 볼록 튀어나올수록

뒷광대축소 수술의 효과가 좋아집니다.

왼쪽 수술 전 사진입니다.

남자분들 중에 뒷광대가 발달된 분들이 많으십니다.

사진에서 정도로 뒷광대가 나오신 분들은

입안절개를 통해서

고전적인 방식으로

광대축소술을 한다고 해도

저 볼록한 뒷광대는 없어지지 않습니다.

반드시 따로 뒷광대부위를

줄여주어야 합니다.

수술결과를 보겠습니다.

변형15분광대축소술 + 뒷광대축소술 + 심부볼지방이식

수술 후 3주째입니다.

심부볼 지방이식은 1차만 받으신 상태이며,

이후 한번 더 터치받으실 예정입니다.

아직 잔붓기가 남아 있으며,

이후 1년까지 잔붓기가 더 빠지면서

지속적으로 얼굴선이 부드러워지게 됩니다.

뒷광대가 상당히 많이 들어간 것을

확인할 수 있습니다.

우측입니다.

역시 광대뼈가 많이 들어갔고,

뒷광대도 정리되면서 울퉁불퉁했던 얼굴선이

깨끗하게 정리된 것이 보입니다.

도움이 되셨길 바랍니다.

결과로 말하는 로코코성형외과입니다.
