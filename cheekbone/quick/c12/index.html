사진으로 알아보는 타입별 광대축소술의 효과

김상호 원장 · 15분 광대축소술 · 로코코성형외과

사진으로 알아보는 타입별 광대축소술의 효과

안녕하세요.

로코코성형외과 김상호원장입 니다.

이전 그림을 통한

광대형태의 분류에 대한 글을 읽으셨는지요.

오늘은 사진을 통해서

다시 한번 type A, B, C 형태를 알아보고,

그 외에 꼭 확인해야 하는

특별한 case 들에 대해 정리해볼까 합니다.

광대의 형태는 크게 나눠서

(1) Type A (효과 좋은 광대)

- 두상보다 광대가 튀어나온 경우

(2) Type B (중등도 효과)

-두상보다 광대가 나오지는 않았지만,

아래쪽에 공간이 있어 효과를 보는 경우

(3) Type C (적은 효과 또는 효과 못봄)

-두상에 비해 광대가 같은 라인 혹은 안쪽에 있고

아래쪽 턱라인과도 너무 부드럽게 연결되는 경우

로 나눌 수 있습니다.

이 분류는

제가 오랫동안 많은 경험을 바탕으로

개인적으로 나누어 본 분류입니다.

각각의 분류 내에서도

좀 더 좋은 효과를 볼 수 있는 경우가 있고,

적은 효과를 보는 경우가 있을 것입니다.

그리고 상기 분류 외에

특별한 형태의 광대가 있습니다.

일단 순서대로 사진을 보시겠습니다.

(1) Type A (효과 좋은 광대)

- 두상보다 광대가 튀어나온 경우

자 사진을 보시면 대략 느낌이 오시나요?

두상(머리뼈)를 기준으로 생각하시고

두상라인(관자가 아님!!!) 보다

볼록하게 많이 튀어나올 수록

많은효과를 보실 수 있답니다.

상담을 하다 보면 많은 분들이

전혀 광대가 튀어나오지 않았는데

본인의 광대를

위의 사진처럼 볼록하다라고 생각하신답니다.

사진을 보여드리고

거울로 본인의 얼굴을 보여드리면서

그 차이를 말씀드려도

"나도 저만큼 나왔는데.. "

라고 인정하지 않으시는 경우도

의외로 많답니다. ;;;

(2) Type B (중등도 효과)

- 일명 아랫광대(제가 붙인 별명입니다. )

두상보다 광대가 나오지는 않았지만,

아래쪽에 공간이 있어 효과를 보는 경우

(3) Type C (적은 효과 또는 효과 못봄)

- 두상에 비해 광대가 같은 라인 혹은 안쪽에 있고

아래쪽 턱라인과도 너무 부드럽게 연결되는 경우

위의 그림에서 우측 그림의 얼굴형을 제가

<도토리형 광대> 라고 이름붙였습니다.

이런 얼굴형을 가진 분들도 상담을 많이 오신답니다.

효과가 매우 적은 얼굴형이랍니다.

도토리형 얼굴을 가진 분들의 사진을 보겠습니다.

대표적인 경우라고 볼 수 있겠습니다.

관자쪽이 넓어서

이런 경우에 헤어라인 모발이식을 하시면

도움이 될 수 있습니다.

우측 사진은 도토리형 광대 중에서도

그나마 약간 광대가 나온 부분이 보이시지요?

Type B-C 중간 정도가 되겠습니다.

그럼 이런 얼굴형을 가진분은

효과를 아예 못보나요?

라고 물어보시면,

흠.. 일단 수술하면

광대뼈는 안쪽으로 밀려 들어갑니다.

그런데 뼈가 들어가는 양에 비해

겉으로 보이는 변화량이 적습니다.

그래서 상담시에 효과가 적을 것이라고

객관적인 설명을 드리고,

나머지는 수술받으시는 분께서

선택하시도록 안내해드리고 있습니다.

이런 얼굴형을 가지신 분들 중에서도

아주 작은 변화에 만족하시는 경우도 있기 때문에

수술을 권할지 말지 의사로서도

애매한 부분이 있답니다.

(장사를 잘하는 의사라면

무조건 수술을 권해야겠지요 )

저는 최대한

객관적인 결과에 대해서만 설명드립니다.

자 오늘은 특이형태의 광대 중

한가지를 보겠습니다.

상담을 하다보면

일주일에 3-4명 정도는 꼭 만나는 형태의 광대입니다.

제가 < W-광대 > 라고 이름붙였습니다.

## W - 광대

옆광대 아래쪽에서 45도 광대 아래쪽으로 연결되는

깊은 그림자가 팔자주름과 만나서

V -자 그림자를 만들고,

양쪽에 V- 모양이 합쳐서

W-형태로 보이는 상태입니다.

이 광대의 특징은

광대뼈가 아닌 볼살(눈 아래쪽 앞볼)이

광대뼈로 인식된다는 점으로,

쉽게말해 광대뼈를 아무리 줄여도

볼살이 광대처럼 보이는 느낌이 남아서

여전히 광대가 많이 남아있는 것 처럼

보인다는 점이랍니다.

다시 말해서

광대뼈가 아닌 <살로 된 광대> 가 있다라고

생각하시면 되겠습니다.

자 대충 그려보았는데요ㅋ

이런 형태의 광대입니다.

얼굴 가운데 부분 전체가

광대뼈처럼 보이지만

(실제 <광대> 라고 인식되는 부분이지요)

<광대뼈>는 붉은색으로 칠한 부분이랍니다.

아래쪽에 파란색 부분은 그냥 <볼살> 입니다.

광대뼈가 아닌데 붉은색과 파란색 모두가

<광대> 로 보이는 형태입니다.

- 그림이 꼭 태극기처럼 되어버렸네요. -_-;;

태극광대...

파란색 부분은 <광대뼈 - 관골>이 아닌

<상악골> 이라는 다른 뼈가 있는 부위입니다.

그렇다고 이 상악골이 돌출되어서

저렇게 광대처럼 보이는 것도 아니랍니다.

사진에서 보이는

형태의 광대가 W- 광대입니다.

본인의 얼굴이 위와 같은 형태이신지

한번 확인해보세요.

잘 모르시겠으면

병원으로 상담오시면

제가 자세히 봐드리고 설명드린답니다.

자, 그러면 위와 같은 W-광대의 경우

어떻게 해결할 수 있을까요?

그건 다음편에

다시 알려드리도록 하겠습니다.

15분 광대축소술에 대해 더 궁금한 점이 있으신가요?
