Type B 광대에서 변형15분광대축소술 ( 퀵광대 + 45도광대축소 )

+ 아큐스컬프 광대라인 지방흡입

+ 로코코 미니리프팅

안녕하세요.

결과로 말하는 로코코~!

김상호원장입니다.

제가 오랫동안 광대축소술을 집도하면서

한국인에서 광대의 형태를

세가지로 분류하였는데요.

위의 글을 참고하시면 도움이 되실 것입니다.

Type B 아랫광대 case 를 보겠습니다.

위의 사진에서

광대뼈는 거의 다 드러나 있습니다.

하지만,

이 사진만 보아서는

광대가 특별히 톡 튀어나와 보이지는 않으시지요?

두상의 폭에 비해

광대돌출은 거의 없는 상태입니다.

턱선까지 보시면

이제는 광대가 나와 보이시지요?

이런 형태를 가진 광대를

제가 type B 광대

다른말로 아랫광대라고

이름 붙였습니다.

이런 광대분류는

나름대로 의미가 있어서 논문도 쓰고,

학회 발표를 해볼까 생각했지만..

아직은 학회 발표가

체질에 맞지 않아서

그냥 블로그로 공개하였답니다.

위의 사진에서 광대는

type B 중에서 아랫공간이 많은 형태라

이런 광대를 가진 분의 경우는

어느 정도 얼굴선이 부드러워지는

효과를 보실 수 있답니다.

하관이 많이 좁은 경우는

심부볼 지방이식을 같이 하신다면

더 좋은 결과를 만들수도 있습니다.

수술전 아래쪽에서 본 사진에서

옆광대 아래쪽에 공간이 보입니다.

광대의 형태와 관계 없이

모든 분들께 변형 15분광대축소술 ( 퀵광대축소술 + 45도광대축소 ) 을

권해드리고 있습니다.

수술 전후 사진을 보겠습니다.

변형15분광대축소술 ( 퀵광대 + 45도광대축소 )

+ 아큐스컬프 광대라인 지방흡입 +

로코코 미니리프팅 수술 후 2주째 사진입니다.

아직 2주째라 수술 붓기가 아래로 내려와서

턱선이 약간 볼록해진 상태입니다.

광대가 잘 들어가서

얼굴선이 부드러워진 것이 보입니다.

얼굴 전체사진을 보면

훨씬 더 변화가 많이 된 것을 볼 수 있답니다.

수면마취로 짧은 시간 수술해서

이 정도의 결과를 만든다는 것은

정말 멋진!! 일입니다.

제가 하는 수술이라서

이렇게 말하는 것이 아니라

진심으로

광대뼈 작아지는 법 중에서

가장 좋은 수술법이라고 생각합니다.

변형15분광대축소술 ( 퀵광대 + 45도광대축소 )

+ 아큐스컬프 광대라인 지방흡입 +

로코코 미니리프팅

수술 후

2주째 아래쪽에서 본 사진입니다.

광대가 잘 들어가서

옆광대 아래쪽 공간이

줄어든 것을 확인할 수 있습니다.
