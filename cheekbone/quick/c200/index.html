15분광대축소술 ( 퀵광대 ) + 로코코 미니리프팅

안녕하세요.

도와줘요 로코코~ 김상호원장입니다.

요즘 로코코에서

광대축소술을 받으시는 분들은

대부분 < 변형15분광대축소술 ( 퀵광대 + 45도광대 축소 )

+ 아큐스컬프 광대라인 지방흡입

+ 미니리프팅 > 수술을 받으시는데요,

가끔은 15분광대축소술 ( 퀵광대 )

만 받으시는 분들도 계십니다.

멍이나 붓기를

최대한 줄이길 원하시는 경우에는

15분광대축소술 ( 퀵광대 ) 수술이 도움이 되는데요,

변형15분광대축소술 (퀵광대 + 45도광대)에 비해서

아무래도 결과라 조금 모자라기 때문에

상담시에는 모든 분들께

변형15분광대축소술을 추천드리고 있습니다.

15분광대축소술 ( 퀵광대 )과

변형15분광대축소술 (퀵광대 + 45도광대)

의 차이점에 대해서

이전에 포스팅했던 글이 있으니 참고하시기 바랍니다.

타병원에서

퀵광대를 받으신 분들과

로코코에서 15분광대축소술 (퀵광대)를

받으신 분들이

나중에 붓기가 빠진 후 약간의 아쉬움 때문에

45도 광대부위를 추

가수술로 줄이는 경우도

상당히 많은 편이랍니다.

그래도 옆광대만 유난히 도드라진 분들에서는

15분광대축소술 (퀵광대) 도 좋은 수술법입니다.

보여드릴 환자분의 수술전 사진을 보겠습니다.

수술 전 정면사진입니다.

type A 광대형태이며

( ) ,

45도 광대아래쪽으로 그림자가 있는 W-광대

( )

가 동반된 상태입니다.

이런 얼굴형을 가진 분들께는

변형15분광대축소술 업그레이드 패키지

1. 변형15분광대축소술 (퀵광대 + 45도광대)

2. 아큐스컬프 광대라인 지방흡입

3. 로코코 미니리프팅

과 함께

심부볼 지방이식을 추천드리고 있습니다.

수술전 아래쪽에서 본 사진입니다.

15분광대축소술 ( 퀵광대 )

+ 로코코 미니리프팅

수술 후 2주째 사진입니다.

Type A 광대이므로

정면에서 옆광대가 잘 들어가

얼굴선이 상당히 부드러워졌습니다.

요즘 옆광대만 넣는 수술은

전체 수술시간이 10분이 걸리지 않는답니다.

15분광대축소술 ( 퀵광대 )

+ 로코코 미니리프팅

수술 후 2주째 아래쪽에서 본 사진입니다.

옆광대가 깨끗하게 들어간 것이 보입니다.

그런데, 앞쪽 45도 광대와의 경계부가 살짝 눈에 보입니다.

타병원에서 퀵광대수술을 받으신 분들이

이 부분이 아쉬워서

45도 광대부위를 다듬기 위해

로코코를 찾아주시는 경우가 많습니다.

그런 경우는 45도 광대축소술

( 필요한 경우는 옆광대 재수술을 같이 시행)

+ 아큐스컬프 광대라인 지방흡입

+ 미니리프팅 으로 수술해드리고 있습니다.

45도만 재수술하는 경우는

이미 옆광대가 꺾여 있는 상태이기 때문에

처음부터 변형15분광대축소술로

수술받는 데 비해서는

45도 광대를 줄일 수 있는 범위가

약간 줄어들어 결과가 조금 못한 편이랍니다.

그래도, 회복기간이

상당히 짧다는 큰 장점을 가지는 수술입니다.
