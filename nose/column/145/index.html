콧대 시작점을 내려주세요

2016. 03. 16

상담을 하다보면 코가 길어보이니

콧대 시작점을 내려달라고 요청하시는 분들이 많으십니다

그런데 많은 분들이 콧대 시작점을 낮추는 방법에 대해서

오해하시는 부분이 있어 제가 간단히 설명드리겠습니다

SE3-TEXT { 제가 코성형을 처음 집도하던 때는위의 그림에서처럼 쌍커풀 높이를 콧대 시작점으로 잡았습니다

하지만 요즘은 위의 그림처럼 대략 눈동자 위치를 콧대 시작점으로 잡고 있습니다 콧대 시작점이 낮아지면 아무래도 코가 짧아 보이기 때문에 좀 더 어려보이고 세련된 느낌으로 디자인이 가능합니다 하지만, 원래 코가 짧은 경우에는콧대 시작점을 높여서 가능한 코가 길어보이도록 디자인합니다 자.. 그러면 콧대 시작점을 내리는 방법에 대해 설명드리겠습니다. 대부분의 환자분들이 실리콘 보형물을 위로 밀어 올리지 말고아래쪽에 위치시키면 콧대 시작점이 낮아진다고 생각하시는 듯 합니다 하지만, 이것은 잘못된 상식(?) 이랍니다

} SE3-TEXT

해골입니다플라스틱으로 만든 모형이니 놀라진 마시고..뼈 모형의 미간에서 코뼈로 이어진 부분이 보이시지요?

실리콘 보형물이 이렇게 콧대 위에 들어가는데요,보형물은 위의 사진에서처럼 끝까지 들어가서 위쪽에 딱 붙어야 합니다

이런식으로 미간쪽에서 떨어지면보형물과 미간 사이에 빈 공간이 생기기 때문에 잘못된 수술이 됩니다콧대 위쪽이 움푹 꺼져 보이거나 체액이 고여서 염증을 유발하겠지요

그래서 일반적으로 생각하시는 것처럼보형물을 아래쪽에 위치시켜서 콧대 시작점을 내리는 것은 아니라는 것입니다그러면 어떻게 콧대 시작점을 내리는가?그것은 일반적으로 콧대 보형물의 두께로 결정이 됩니다보형물이 두꺼울수록 시작점이 높아지면서 코가 길어지게 되고,낮은 보형물의 경우는 시작점이 낮아지면서 코가 좀 더 짧아보이게 됩니다그래서 짧은 코의 경우는 일반적인 경우보다 콧대를 더 높여서 길어 보이게 만들고,긴 코의 경우는 절대 콧대를 많이 높이면 안됩니다

얼굴뼈 모형에서 사람 얼굴은 대략 저렇게 위치합니다

얇은 실리콘을 넣었을 때 A 점이 콧대 시작점이 되구요..

더 두꺼운 실리콘을 넣으면 B점이 시작점이 되어 A점 보다 시작점이 더 위로 올라가게 되는 것입니다말로 설명하면 간단한 내용인데 글로 설명드리려니 쓸데 없이 길어지네요 -_-;정리하면..일반적으로 코의 시작점은 위에 설명드린 것처럼 콧대보형물의 높이로 결정됩니다그 외에 미간쪽의 볼룸을 더 만드는 지방이식이나 필러시술을 하게 되면 코의 시작점을 낮출 수 있답니다그런데, 이런 시술은 극히 낮지만 실명의 위험성이 있으므로 신중하게 의논해서 결정해야 할 것입니다도움이 되셨나요?전 이만 퇴근하겠습니다 편안한 밤 되세용~
